# CS5463 – High Performance Machine Learning
## Term Project

### Title
Sparse Transformers: A Survey of Efficient Attention Mechanisms for Large-Scale Deep Learning

### Student
Thanush Koshekay

### Project Description

The project was initially intended to optimize the use of GPUs to execute machine learning workloads based on the project proposal. Because the most frequent computation in transformer attention mechanisms is the multiplication of matrices, this survey advances the scope to examine efficient transformer architectures and sparse attention mechanisms that could minimise the level of computation and memory.

This questionnaire examines the recent studies on the application of sparse transformer models and efficient attention implementations that can scale deep learning systems. The survey classifies the sparse methods of attention and evaluates their computational efficiency, memory consumption, scaling and performance on the modern GPU architecture.
